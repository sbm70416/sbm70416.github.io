#### 변경이 부가능한 자료형으로 몇가지를 제외하고는 사용법이 똑같다.(주의: 바꿀 수 없다.)
```python
d={'name':'Jane', 'age':12,'number': 123456}

```
### <span style="background-color:yellow">딕셔너리란</span>
#### 중괄호 안에 key:value형식으로 저장
 ```python
a={'a':1,'b':2,'c':3,'d':4}
print(a.keys())
print(a.values())
print(a.items())
a.clear()
print(a)
```
